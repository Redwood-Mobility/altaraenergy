Color Usage Guidelines

  • Current Teal: Primary brand color for logos, headings, key UI elements
  • Ember Orange: Accent color for CTAs, highlights, important notifications
  • Deep Teal: Use for hover states, active states, depth
  • Bright Orange: Use for hover states on orange elements
  • Concrete: Use for subtle text, borders, disabled states
  • Slate: Use for body text, dark UI elements
Accessibility Note

All color combinations meet WCAG AA standards for contrast. Current Teal on white: 5.2:1. Ember Orange on white: 3.8:1 (use larger text). Slate on white: 12.6:1.

Logo

The Altara logo features a circular orbital ring with an "A" symbol at the center. The orbital represents energy flow, connectivity, and continuous motion. The "A" is bold and confident, representing reliability and forward movement.

Primary Logo

Primary logo — Full color with gradient orbit

Minimum Size

To maintain legibility and visual impact:

  • Icon only: Minimum 32px (digital), 0.5 inch (print)
  • Icon with wordmark: Minimum 120px width (digital), 1.5 inches (print)

Clear Space

Maintain clear space around the logo equal to the height of the "A" symbol on all sides. No text, graphics, or other elements should intrude into this space.

Logo Usage Don'ts

To maintain brand integrity, never alter the logo in the following ways:

❌ Don't Rotate
❌ Don't Stretch
❌ Don't Change Colors
❌ Don't Add Effects

Additional Don'ts

  • Don't recreate or redraw the logo
  • Don't add outlines or borders
  • Don't place on busy backgrounds that reduce legibility
  • Don't use the logo smaller than minimum sizes
  • Don't alter the spacing between orbit and "A" symbol
  • Don't rearrange logo elements
  • Don't animate the logo without brand team approval

Print Applications

For print materials, always provide vector files (SVG or PDF) to ensure crisp reproduction at any size. When printing in single color (such as embroidery or screen printing), use the monochrome teal variant.

Pathfinder Network Integration

Altara is a member of the Pathfinder Rewards network. This relationship should be clearly communicated but not dominate the brand presence.

Co-Branding Guidelines

  • Primary branding is always Altara (larger, more prominent)
  • Pathfinder badge should appear secondary (smaller, supporting role)
  • Use "Pathfinder Accepted Here" or "Part of the Pathfinder Network" language
  • Maintain clear space between Altara and Pathfinder marks

Where to Show Pathfinder

  • Website footer with badge/link
  • Charging station signage (secondary placement)
  • Membership materials and app
  • Point-of-charge displays

Where NOT to Show Pathfinder

  • Primary website header
  • Main business cards
  • Canopy primary signage (can be on secondary signs)
  • App icon
Partnership Balance

Altara stands as an independent brand. Pathfinder is a valuable network partner, not a parent brand. The relationship enhances value but doesn't define Altara's identity.

File Format Notes

SVG (Scalable Vector Graphics) is the preferred format for all applications. SVG files scale infinitely without loss of quality and work across web, print, and digital applications.

If you need other formats (PNG, PDF, EPS), export from the SVG files at your required resolution. For print, always use vector formats when possible.
