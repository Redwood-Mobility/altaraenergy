# Altara Energy Website

The official website for Altara Energy - America's first premium EV charging network designed for scenic highway corridors and National Parks routes.

## About

Where charging becomes part of your journey.

This is a Jekyll static site designed for GitHub Pages deployment.

## Brand

- **Primary Color**: Ocean Deep (#0A4B6E)
- **Accent Color**: Sunset Coral (#FF6B4A)
- **Tagline**: "Where charging becomes part of your journey"

## Development

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# Build
bundle exec jekyll build
```

## Structure

- **index.html**: Home page with hero, core promises, and hub-satellite model
- **about.html**: Company story, mission, and values
- **locations.html**: Network locations (Basecamps and Waystations)
- **membership.html**: Pathfinder Rewards program with 4 tiers
- **contact.html**: Contact form and partnership inquiries

## Images

Place brand assets and page images in the `/images` directory:
- Hero images (scenic highways, golden hour)
- Logo files (SVG preferred)
- Location photos
- Membership tier visuals

## Deployment

This site is configured for GitHub Pages deployment.
